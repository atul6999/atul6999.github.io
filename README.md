# atul6999.github.io

Static GitHub Pages resume website for Atul Mishra.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static web server.

## Publishing

Push this repository to GitHub. GitHub Pages will publish it at:

https://atul6999.github.io

The downloadable public resume is stored at `assets/Atul_Mishra_Resume.pdf`.
It intentionally omits direct email and mobile details.
